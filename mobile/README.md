# YOU Mobile

React Native and TypeScript app for YOU.

## Start

```bash
npm install
npm run start
```

Run a platform:

```bash
npm run android
npm run ios
```

## Folder Guide

| Folder | Purpose |
| --- | --- |
| `src/assets` | Images, fonts, and static assets |
| `src/components` | Shared UI components |
| `src/screens` | Feature screens grouped by product area |
| `src/navigation` | App navigators and route types |
| `src/services` | API clients and external integrations |
| `src/hooks` | Reusable React hooks |
| `src/context` | React context providers |
| `src/store` | Future app state store |
| `src/utils` | Pure helpers |
| `src/constants` | Static app constants |
| `src/theme` | Colors, spacing, typography |
| `src/types` | Shared TypeScript types |

## API

The placeholder API client points to:

```text
http://localhost:5000/api/v1
```

Use `http://10.0.2.2:5000/api/v1` when calling your local machine from an Android emulator.
